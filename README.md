# n8n AutoMator — Shopify App Home

This is a simple static app home page for the private Shopify app used by Madeleine’s Haus.

The app connects Shopify with n8n for:

- Blog publishing
- Image automation
- Shopify Admin API workflows

This page is informational only. It does not contain API tokens, client secrets, OAuth codes, workflow URLs, or private configuration.
